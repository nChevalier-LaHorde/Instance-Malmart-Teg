 EH3Anchor

enum EH3Anchor

Anchor flag. A full anchor is a bitwise OR of a horizontal and a vertical anchor flag, ie. (A_Center | A_Bottom)

 EH3ColliderDynamicsType

enum EH3ColliderDynamicsType

Collider dynamics type. This influences how the physics engine considers the collider.

Enumerator
CDT_Static 

Static collider (will never move)

CDT_Kinematic 

Kinematic collider (can be moved by code)

CDT_Dynamic 

Dynamic collider (can be moved by code or by the physics engine)
